Clock Clock

Group of analog clocks creating a single digital time display.

category

Art, Code

date

Fall 2014

technologies used

Processing 2

course

Electronic Media Studio: Interactivity

This clock is a digital recreation based on an inspirational video by "Humans Since 1982". It consists of 24 analog clocks representing sections of one, large digital-looking clock display, as well as 2 smaller 60-second analog clocks representing the colon symbol between the hours and minutes.

You can view the original project documentation on the class website, or download and run the MacOS compiled .app.

Every 30 seconds an animation of all clocks are triggered, lasting 15 seconds. The second animation consists of a transition to the next minute.

The biggest challenge was defining the behaviors of each clock, its start and finish positions, and the speed at which they must rotate in unison. They also had to have a level of "chaos," with each rotating different directions or speeds than neighbors.

In this picture, I'm planning the directions and speeds of each clock.

Another challenge was digitally displaying the clocks, which involves writing the calculations for the angles and circular animations of the clock hands.
