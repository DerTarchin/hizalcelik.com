Nickel

Automatically discovers trading opportunities in the stock market.

category

Code, UI/UX Design

date

2017 (Ongoing)

technologies used

Python, Django, HTML5, CSS3, Javascript, jQuery, KiBot API, Intrinio API, Heroku

client

TradeTheTrend

Nickel is a service that automatically finds trading opportunities in the stock market based on trends, charts and technical analysis. My algorithm scans thousands of stocks in the market and analyzes their charts based on a modular set of patterns, trends or other indicators and displays the results for users to use in their daily trading decisions.

I put a great emphasis on design and user experience, something that is sadly rare in the financial industry. I am also constantly optimizing my algorithms and front end code to be more efficient and speedy.

The provided results by Nickel are displayed in a modern, clean design with a minimal interface, giving users a clear view of the pattern while not overloading with unnecessary information.

Unlike traditional stock screeners that only use mathematical indicators and single-bar patterns, Nickel can also find multi-bar chart patterns and any other visual indicator. The list of patterns and indicators used by my algorithms is also expandable and modular.

Users are able to browse different patterns indicated by minimal icons in the menu bar. The app also provides the ablity to filter, sort and save stocks based on a number of parameters, as well as the ability to export their lists to the popular stock charting app TradingView, which I've also made a matching CSS dark theme for.

Nickel is an ongoing project in collaboration with TradeTheTrend. Due to API limitations at the moment, Nickel is only able to scan markets once a day, after markets close. While this is very useful for trading the next day, eventually I would love to add real-time scanning ability with custom notifications and alerts for users to take advantage of. I plan to use Intrinio for real-time market data.
