Chirp

A beginner's attempt at 3D deformers and modeling.

category

Art, Animation, Video

date

Spring 2016

technologies used

Maya, Vue xStream, Premiere Pro

course

Animation Art and Tech

Final Video

Chirp is a short technical experiment on basic 3D modeling, environment creation, and using 3D deformers. Although the assignment this was for only wanted us to use 3D deformers, I wanted to push myself to explore Vue xStream software and modeling my own basic bird, too.

Designing the Bird

Instead of downloading a model like I did for the tree, I wanted to make the bird from scratch using my limited 3D modeling knowledge. I wanted the bird to be simple and cute, inspired by the short For the Birds by Pixar Studios.

Environment

The software Vue xStream allows an artist to create any shape or form and apply photorealistic textures to it, making this scene look more like a photo than a 3D render.
