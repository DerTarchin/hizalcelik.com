CSS Dark Themes

CSS styles designed to transform a site into a dark theme.

category

Design, Code

years

2017 to present

technologies used

userstyles.org, Stylus

download links

TradingView, Stocktwits

These are some CSS styles intended to transform a website into a dark theme, often times also improving the UI to look more modern and stylish. The code I write is then distributed on the userstyles.org platform (somewhat defunct as of 2020) and applied to a website using an extension called Stylus. Some of these styles were initiall started as a project for TradeTheTrend.

Unfortunately as websites are prone to changing every few years, some of these styles no longer work. And to make things worse, userstyles.org seems to be defunkt and no longer maintained as of 2020, which makes it more difficult to successfully upload updates to my styles.
