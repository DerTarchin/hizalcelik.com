Instruction Experiment

Seeing how different people follow the same instructions.

category

Art

date

Fall 2014

course

Electronic Media Studio: Interactivity

This is an experiment on writing and following instructions. The process is simple: write a "looping" instruction to draw something, then have several volunteers attempt to follow it without seeing an example. It is surprising how many variations you can get from even the strictest instructions.

You can view the original project documentation on the class website.

                        
1.  
Use two colors.
Line colors do not matter, but circle colors do.

2.  
Draw three separate lines.

3.  
Draw circles connected to ends of each line, with a different color on each end of each line.

4.  
You may not cross over any existing lines when drawing.
If possible, connect each circle to two circles of different color from itself.

5.  
Choose an existing color.

6.  
While: you have space/patience, {
    - you may not cross over any existing lines while drawing.
    - draw a line from the *midpoint* of any existing line 
    - draw a circle on end of line
    - if possible, connect circle to two other circles of a different color from itself
    - switch colors and repeat step 6
}

7.  
Thank you for participating.

Instructions

Reflections on Responses

The second image (last to be created) was the most shocking and different. I had imagined my instructions to be clear and result in the other two images, but I never imagined the drawer would read into the directions and really overthink it and think outside the box like that. No rules were broken- I never stated the circles shouldn’t be connected to each other physically by its outline, and I never stated the circles have to be a single color. Although I know that I would revise those rules later to have a more set design standard.
