Color Study: Saturation

A study on saturation within a limited value spectrum.

category

Art, Code

date

Spring 2017

course

Color Theory

The purpose of this study is to use saturation and hue effects to create lots of spatial depth within a restricted value range. That means, having a variety of shades and colors, but if converted to black and white, it would look like a uniform gray.
