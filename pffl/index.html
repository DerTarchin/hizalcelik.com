client

Pittsburgh Flag Football League

The PFFL Scheduler* is a schedule and match generator for Pittsburgh Flag Football League, and is adaptable to be used by any football league. Scheduling matches is a time consuming and complex task, as there are several rules and conditions for the number of times each team of each rank or division can be matched with another. The purpose of this software was to apply all those complex rules into an automated application, saving time and effort.

*This project was not completed, all screenshots are from mid-development documenation.
